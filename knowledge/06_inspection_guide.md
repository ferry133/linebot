# 室內裝修驗收完全指南

## 驗收的重要性

驗收是消費者最後一道把關機會：
- 發現施工缺失，要求業者免費修繕
- 確認建材與合約規格相符
- 尾款支付的前提條件

> **原則：驗收通過才付尾款**，這是消費者最重要的自保手段。

---

## 驗收類型

### 1. 階段性驗收（施工中）
- 不能只等完工後才驗收
- 隱蔽性工程封牆前必須驗收（水電管線、防水層）
- 每個工種完成後確認，才進行下一工種

### 2. 完工驗收
- 全部工程完成後進行全面檢查
- 帶著**驗收清單**逐項勾選
- 發現問題貼標籤、拍照、記錄

---

## 隱蔽性工程驗收（最重要）

### 水電管線（封牆前）
- [ ] 確認插座位置與設計圖一致
- [ ] 確認電路迴路數量（開關面板標示清楚）
- [ ] 確認熱水管有包覆保溫材料
- [ ] 確認排水管坡度（目測確認不積水）
- [ ] 確認弱電線路（網路、電視、監視器）預埋完成
- [ ] **拍攝管線走向照片存檔**（日後修繕鑽孔定位用）

### 防水工程（磁磚貼覆前）
- [ ] 浴室地板與牆面（至少 180cm 高）防水層施作
- [ ] **盛水測試**：蓄水至少 2–3cm，等待 24–48 小時，確認樓下無滲漏
- [ ] 陽台、廚房有積水可能的地方

---

## 完工驗收清單

### 牆面與天花板
- [ ] 油漆是否均勻（無刷痕、色差、漏漆）
- [ ] 牆面是否平整（無裂縫、鼓起）
- [ ] 天花板接縫處是否有開裂
- [ ] 壁紙是否有氣泡、翹邊、拼花錯誤
- [ ] 燈槽邊緣是否整齊

### 地板
- [ ] 地板是否平整（用長板尺測試，不超過 3mm）
- [ ] 踩踏是否有異音（空鼓）
- [ ] 地板與牆面收邊是否密合
- [ ] 不同材質交接處收邊條是否牢固
- [ ] 地板顏色、材質與選料確認書一致

### 木作工程
- [ ] 所有門是否開關順暢（不卡頓、不自動關）
- [ ] 抽屜開關是否順暢
- [ ] 門縫是否均勻（上下左右各約 3mm）
- [ ] 把手、鉸鍊是否牢固
- [ ] 木作表面是否有刮痕、色差

### 磁磚
- [ ] 磁磚是否有空鼓（用硬幣敲擊，聲音空洞代表空鼓）
- [ ] 磁磚縫隙是否均勻
- [ ] 填縫是否完整（無缺縫、裂縫）
- [ ] 磁磚邊角是否有缺角、破損

### 水電
- [ ] 所有插座通電測試
- [ ] 所有燈具亮燈測試
- [ ] 冷氣試機（冷暖兩用確認）
- [ ] 水龍頭冷熱水確認
- [ ] 蓮蓬頭/浴缸水壓測試
- [ ] 排水速度確認（無積水）
- [ ] 確認無漏水跡象

### 衛浴設備
- [ ] 馬桶沖水測試（確認水封正常）
- [ ] 洗手台排水測試
- [ ] 浴室門/淋浴拉門順暢無卡頓
- [ ] 矽利康（矽膠）收邊整齊、無縫隙

### 廚房
- [ ] 廚具門板開關順暢
- [ ] 瓦斯台測試（點火正常）
- [ ] 抽油煙機吸力測試
- [ ] 廚房排水測試

---

## 建材確認查核

- [ ] 確認地板品牌、型號與材料確認書一致
- [ ] 確認磁磚品牌、花色、尺寸與確認書一致
- [ ] 確認系統傢俱板材品牌與等級
- [ ] 索取防火建材相關證明文件（若有使用）
- [ ] 索取水電材料規格表（管徑、電線截面積）

---

## 驗收發現缺失的處理步驟

1. **現場貼標籤標示**缺失位置
2. **拍照存證**（含標籤一起拍）
3. 在**驗收單上記錄**所有缺失（讓業者簽名確認）
4. 約定**修繕期限**（建議不超過 2 週）
5. 缺失全數修繕後，再次驗收確認
6. 確認通過後，支付尾款

---

## 驗收後應索取的文件

| 文件 | 說明 |
|------|------|
| 保固書 | 載明保固範圍、期限、申請流程 |
| 材料確認書 | 所有使用建材的品牌、型號 |
| 水電管線照片 | 隱蔽工程配置記錄 |
| 施工許可證（若有申請） | 合法施工證明 |
| 設計圖 | 竣工圖紙 |

---

## 常見缺失與處理

| 缺失類型 | 嚴重程度 | 處理方式 |
|---------|---------|---------|
| 防水滲漏 | 高 | 拆除磁磚重做防水，不接受補縫了事 |
| 地板空鼓 | 中高 | 重新鋪設，面積小可局部處理 |
| 油漆色差、漏漆 | 中 | 要求全面重新上漆 |
| 木作門不順暢 | 中 | 調整鉸鍊或刨除 |
| 磁磚空鼓（非浴室） | 中 | 小面積可填充，大面積需重鋪 |
| 輕微刮痕 | 低 | 修補即可 |

## 驗屋服務（可考慮委託）

若不熟悉施工品質判斷，可委託**驗屋公司**進行專業驗收：
- 費用：約 3,000–8,000 元
- 提供書面驗收報告
- 使用儀器檢測（如：紅外線熱像儀偵測漏水、水平儀測量平整度）
- 適合高單價裝修或對施工品質有疑慮者
