# 室內裝修費用行情與預算規劃

## 每坪費用行情（2025–2026 台灣市場）

### 依屋況分類

| 屋況 | 每坪費用範圍 | 特點 |
|------|------------|------|
| 新成屋（輕裝修） | 4–7 萬元 | 基礎設施較新，主要做美觀裝飾 |
| 新成屋（一般裝修） | 6–12 萬元 | 含固定傢俱、木作 |
| 中古屋 / 舊屋翻新 | 8–15 萬元 | 需更換水電管線、泥作較多 |
| 全室高端設計 | 15–20 萬元以上 | 名師設計、進口建材 |

### 依設計風格分類

| 風格 | 費用特性 |
|------|---------|
| 現代簡約 | 工序相對簡單，費用較低 |
| 北歐 / 工業風 | 建材要求較高，中等費用 |
| 新古典 / 歐式 | 線板、造型多，費用偏高 |
| 日式侘寂 / 無印風 | 建材質感要求高，費用中高 |

> 雙北地區（台北市、新北市）因租金與人力成本，裝修費用較其他縣市**高 15–20%**。

## 費用組成結構

### 主要費用項目

| 項目 | 費用占比 |
|------|---------|
| 室內設計費 | 5–15%（依設計師收費方式） |
| 工程費（含所有工種） | 55–65% |
| 家電設備 | 約 15% |
| 家具 | 約 15% |
| 軟裝（窗簾、燈具、裝飾） | 5–10% |

### 設計費計費方式

| 方式 | 說明 |
|------|------|
| 工程比例抽成 | 工程費 × 10–15% |
| 每坪計費 | 約 3,000–6,000 元/坪 |
| 固定費用 | 依案件複雜度議定 |
| 包含於工程費 | 設計費內含（統包模式較常見） |

## 預算分配建議

### 建議分配比例（以 100 萬元總預算為例）

```
工程費          60–65 萬
家電設備        15 萬
家具            15 萬
軟裝家飾        5–10 萬
備援預算（10%） 10 萬（需額外預留）
```

### 建議備援預算

- 裝修總預算至少保留 **10% 備援**
- 舊屋翻新建議保留 **15–20%**（易出現隱藏問題）
- 常見追加項目：隱藏性壁癌處理、管線老化更換、增設插座

## 省錢策略

### 合理省錢方式

1. **局部裝修**：只翻新需要改變的區域，保留堪用的木作
2. **系統傢俱取代木作**：系統板材工廠量產，比木工現場製作便宜
3. **國產磁磚取代進口磚**：品質差距縮小，價格差 30–50%
4. **自購建材**：屋主自行採購燈具、衛浴五金，可節省業者代購加成
5. **比價**：同一份需求找 2–3 家報價，比較「細項」而非只比總價

### 不建議省的項目

| 項目 | 原因 |
|------|------|
| 防水工程 | 做不好後續維修費用是原本數倍 |
| 水電工程 | 隱藏在牆內，出問題難以修繕 |
| 板材防潮防火等級 | 影響居住安全與健康 |

## 付款節點建議

| 階段 | 付款比例 |
|------|---------|
| 簽約定金 | 不超過總工程款 **10%** |
| 開工款 | 不超過總工程款 **20%** |
| 工程中期（進度約 50%） | 30–40% |
| 完工驗收後 | 最後 30–40% |

> 2026 年草案草擬中：限定金不逾 10%、開工款不逾 20%，將成為法定保護。

---

## 費用透明度建議

簽約前要求對方提供**詳細工程估價單**，應包含：
- 施工項目（逐項）
- 材料品牌、型號、規格
- 數量與單位
- 單價與小計

「總價制」估價（只給總數不細列）是糾紛的溫床，應避免接受。
