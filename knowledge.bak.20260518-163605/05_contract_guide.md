# 室內裝修合約簽訂完全指南

## 為什麼合約至關重要

- 裝修金額動輒數十萬到百萬，合約是保障雙方權益的唯一書面依據
- 無合約糾紛，消保單位難以介入
- 口頭承諾不具法律效力，一律要求**書面化**

---

## 合約應包含的核心條款

### 1. 工程範圍（Scope of Work）
- 詳細列出每一項工程，逐項說明
- 標示材料品牌、型號、規格、顏色
- 確認「含」與「不含」的項目（如：廢棄物清運是否包含）

### 2. 工程金額與計費方式
- 總工程款（應為固定金額或明確計算公式）
- 各工項的單價、數量、小計
- 追加工程的議價規則（需書面確認才生效）

> 警示：只給總價不列細項的報價，是日後糾紛的主要來源，**應拒絕接受**。

### 3. 施工時程
- 開工日期、預計完工日期
- 各工種進場時間表
- 延誤罰則（如：每逾期一天罰款千分之一）

### 4. 付款節點

| 付款時機 | 建議比例 |
|---------|---------|
| 簽約定金 | 不超過 **10%** |
| 開工款 | 不超過 **20%** |
| 工程中期（約 50% 進度） | 30–40% |
| 完工驗收後 | 最後 **30–40%**（尾款） |

> 重要：尾款應於**驗收通過後**再支付，是消費者最重要的一道保護。

### 5. 保固條款
- 保固期起算日（應為驗收合格日，非完工日）
- 保固期限（建議要求 **至少 1 年**）
- 各工項保固範圍（木作、泥作、水電分開列明）
- 保固排除項目（如：人為損壞、天災）
- 保固申請流程與回應時限

### 6. 設計圖說
- 設計圖應作為合約附件
- 明確約定「設計圖即為施工依據」，**禁止「僅供參考」字樣**
- 材料選用表作為驗收依據

> 2025年11月預告的定型化契約草案，將禁止業者使用「設計圖僅供參考」條款。

### 7. 爭議解決機制
- 指定仲裁或調解機構
- 雙方管轄法院約定

---

## 常見合約陷阱

| 陷阱 | 風險 | 應對方式 |
|------|------|---------|
| 「設計圖僅供參考」 | 完工可能與預期大相徑庭 | 要求刪除，並要求圖說作為合約附件 |
| 無詳細工項列表 | 日後追加費用爭議 | 要求逐項報價單 |
| 「如遇材料漲價業者有權調整」 | 費用無上限 | 約定固定價或設定漲幅上限 |
| 保固未寫清楚範圍 | 業者拒絕保固理賠 | 要求列明各工項保固年限 |
| 定金超過 10% | 付款後業者跑路損失大 | 堅持定金不超過 10% |
| 增項未書面化 | 業者事後追加費用 | 任何追加必須雙方簽字書面確認 |
| 完工定義不明確 | 業者提前索取尾款 | 合約載明「驗收合格」才算完工 |

---

## 合約簽訂前的審閱期

- 合約應至少提前 **5 天**給予消費者審閱（預告草案規定，現行建議適用）
- 不應在現場當場逼簽
- 可攜回請律師或有經驗的朋友協助審閱

---

## 合法業者查詢

- 至**全國建築管理資訊系統入口網**查詢業者登記狀態
- 確認業者有室內裝修業登記證
- 合約業者名稱應與登記名稱一致

## 政府資源

| 資源 | 用途 |
|------|------|
| **1950** 消費者服務專線 | 消費糾紛申訴 |
| 內政部營建署合約範本 | 室內裝修設計合約書範本，可免費下載作參考 |
| 各縣市消保官 | 調解裝修糾紛 |
| 行政院消費者保護委員會 | 重大消費糾紛 |

---

## 糾紛處理流程

```
發現問題 → 書面通知業者（留存紀錄） → 協商修繕期限
    ↓ 若業者不處理
撥打 1950 申訴 → 消保官介入調解
    ↓ 若調解失敗
民事訴訟（建議委任律師）
```

**重要**：發現問題時，**先拍照存證**，再口頭溝通，最後書面通知。不要在未留存證據前自行修繕，會喪失求償依據。
