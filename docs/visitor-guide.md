# 意念情境 LINE Bot — 歡迎！新朋友使用說明

> 給剛加入、尚未開通的朋友（角色：visitor）。歡迎認識「意念情境室內裝修」🙂

---

## 1. 你現在就可以問

直接打字問機器人，關於：

- **服務項目**：全室裝修、局部改造、木作、水電、地板磁磚等
- **施工流程、報價方式**怎麼進行
- **預約現場勘查**、一般裝修諮詢

---

## 2. 想查「工程進度」？需要先開通

查看實際工程進度、排程、付款狀態，是給**已合作的客戶／配合廠商／內部員工**的功能。你目前是「**訪客**」，還沒有查詢權限。

如果你是：
- **我們的客戶（屋主）** → 請告知你的**案場／姓名**，我們幫你綁定後即可查你的工程。
- **配合的工班／廠商** → 請聯繫對接窗口，開通後即可收到你負責工項的提醒。
- **公司同仁** → 請找管理員開通員工身分。

---

## 3. 怎麼開通

在這個聊天室留言告訴我們你的**身分與案場**，或直接聯繫你的窗口；我們確認後就會幫你開通。開通後再點下方「📖 使用說明」，就會看到**屬於你角色**的完整操作說明。

---

## 4. 小提醒

- 機器人提供的是一般諮詢，**報價與日期以現場勘查與合約為準**。
- 需要專人服務時，隨時說「請專人聯繫」即可。

> 很高興認識你，有任何裝修問題都歡迎在這裡問我們 ✨
